SYL — START HERE
================

This is SYL (Space You Land), a from-scratch, 100%-realistic space game built in
Unreal Engine. (It began under the old name "Curtis AI Unreal Lab.")

There are only two things to know:

  TO LAUNCH
  1. Open the Unreal project: double-click CurtisAILab.uproject (UE 5.8).
     The MCP server auto-starts (Edit > Editor Preferences > Model Context
     Protocol > Auto Start Server; "Unreal MCP" plugin enabled).
  2. Double-click START_CLAUDE.cmd to open Claude in this folder.

  TO CONTINUE THE BUILD IN A NEW CHAT
  3. Open PASTE_INTO_NEW_CHAT.txt, copy ALL of it, and paste it as your first
     message. The new session reads SESSION_HANDOFF.md + BUILDLOG.md and picks up
     exactly where the last one left off.

WHAT EACH FILE IS (so nothing is confusing)
  PASTE_INTO_NEW_CHAT.txt  -> the prompt you paste into a brand-new chat
  SESSION_HANDOFF.md       -> how the AI operates (tools, pipeline, gotchas)
  BUILDLOG.md              -> current state, full history, and what's next
  START_CLAUDE.cmd         -> launcher: opens Claude in this folder
  CurtisAILab.uproject     -> the Unreal project file (double-click to open UE)
